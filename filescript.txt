Percobaan untuk melakukan pull dan push di git dan github
1. bukalah terminal atau command prompt
2. membuat directory: 0147-MIL
3. melakukan git init
4. coba membuat file .txt
5. melakukan git add .
6. melakukan git commit -m "belajar commit"
7. git remote add origin master ...
8. upadate filescript
9. lakukan add, commit, git push
